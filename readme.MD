### question && answer
  wordcloud 直接用pip安装可能报错，可以去https://www.lfd.uci.edu/~gohlke/pythonlibs/#lxml 下载完  pip install xxx.whl
  echarts 使用柱状图 is_convert=True(y轴做为名称)时有个Bug，y轴显示的不是名称而是数字，所以尽量不要用这个属性
